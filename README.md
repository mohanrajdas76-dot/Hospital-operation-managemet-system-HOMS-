# Hospital-operation-managemet-system-HOMS-
The Hospital Operation Management System (HOMS) is a software application designed to manage hospital activities efficiently. It helps in handling patient registration, doctor information, appointment scheduling, medical records, and billing processes in a digital way. 

INSTAL XAMP APP
 STEP 1 : INSTALL THE XAMP THROUGH THE GOOGLE
 STEP 2 : THEN OPEN XAMP APP 
 STEP 3 : START THE APACHE AND MYSQL 
 STEP 4 : GIVEN DATASET AND OPEN THE LOCALHOST FOR CHROME 
 STEP 5: DO IT PASTE THE URL FOR : LOCALHOST:\HOS\
 STEP 6 : AND RUN THE LOADING PAGE FOR THROUGH THE CHROME 
 STEP 7 : OPEN THE PROJECT LOGIN AND HOME PAGE
